Our Mission

Have you ever stared at your closet while running late and had no idea what to wear?

WearCast is here to save your time and help you wear stylish and weather-proof at the same time.

How does it work?

“WearCast” is a two-tier system designed to give the user suggestions for appropriate clothing based on current weather conditions.

Using sensors integrated with a Wio Terminal the system is able to gather raw data of different weather parameters such as temperature and humidity. These sensors continuously gather data on the surrounding environment, which is processed and then transmitted to a website to be displayed as separate sensor values. The system is also connected to a LED sensor that changes colors to red or blue to visually represent the temperature.

By analyzing and displaying these values, the user can access the weather data alongside tailored clothing suggestions and empowering messages. This ensures that the user is always well-prepared and fashionably attired for any weather scenario.
